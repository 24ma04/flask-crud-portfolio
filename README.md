# Flask CRUD アプリ（ポートフォリオ）

## 概要
このアプリは Flask で作成した CRUD（Create, Read, Update, Delete）アプリです。
動的にカードを追加・編集・削除できます。

## 機能
- カードの一覧表示
- 新規カードの追加
- 既存カードの編集
- カードの削除
